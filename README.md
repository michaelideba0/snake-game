# 🐍 Snake Game

A fun and classic **Snake Game** built using **HTML, CSS, and JavaScript**.  
Playable online directly through your browser or on GitHub Pages.

---

## ✨ Features
✅ Arrow key or swipe control  
✅ Growing snake  
✅ Score tracking  
✅ Game-over and restart system  

---

## 🛠️ Technologies Used
- HTML5 Canvas  
- CSS3  
- JavaScript (ES6)
